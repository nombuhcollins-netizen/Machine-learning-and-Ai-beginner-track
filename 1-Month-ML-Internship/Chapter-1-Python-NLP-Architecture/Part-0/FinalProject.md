# Final Project: Text Analysis Engine

**Duration:** 2-3 Hours | **Difficulty:** Intermediate | **Goal:** Apply all Part-0 concepts

---

## Project Overview

Build a command-line text analysis tool that processes text files and provides statistics.

## Requirements

### Part 1: File I/O & Data Processing
- Read a text file
- Split text into words and sentences  
- Handle errors gracefully

### Part 2: Data Structures
- Create frequency dictionaries
- Use sets to find unique words
- Organize data in nested structures

### Part 3: Analytics
- Calculate statistics (word count, average word length)
- Find most common words
- Identify unique sentences

### Part 4: Text Cleaning
- Normalize text
- Remove punctuation
- Handle edge cases
