# Glossary - Part 0

## A
- **Argument** - Value passed to a function
- **Array** - Collection of items (list, tuple, or NumPy array)

## B
- **Boolean** - Type with True/False values
- **Break** - Statement to exit a loop

## C
- **Concatenation** - Joining strings together
- **Continue** - Statement to skip iteration

## D
- **Dictionary** - Collection of key-value pairs
- **Data Type** - Type of data (int, float, str, bool, etc.)

## E
- **Enumerate** - Function to get index and value

## F
- **Float** - Decimal number type
- **For loop** - Loop that iterates over items
- **Function** - Reusable block of code  

## G
- **Global** - Variable accessible everywhere

## I
- **Index** - Position of an item (starts at 0)
- **Integer** - Whole number type
- **Immutable** - Cannot be changed

## K
- **Keyword** - Reserved word in Python

## L
- **Lambda** - Anonymous function
- **List** - Mutable collection of items
- **Local** - Variable accessible only in function

## M
- **Module** - Python file with reusable code
- **Mutable** - Can be changed

## N
- **None** - Type representing no value
- **NumPy** - Library for numerical computing

## O
- **Object** - Instance of a class

## P
- **Parameter** - Input to a function definition

## R
- **Return** - Send value back from function

## S
- **Set** - Unordered collection of unique items
- **Slice** - Extract part of sequence
- **String** - Text data type

## T
- **Tuple** - Immutable sequence
- **Type** - Category of data

## V
- **Variable** - Named container for data

## W
- **While** - Loop that continues while condition is true
