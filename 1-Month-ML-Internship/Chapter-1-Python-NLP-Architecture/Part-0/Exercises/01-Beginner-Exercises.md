# Beginner Exercises - Part 0

Practice these exercises to master the fundamentals!

## 1. Variables & Types (From Lesson 1)

**Exercise 1: Profile Creator**
Create variables for a person's profile:
- Name (string)
- Age (integer)
- Height (float, in meters)
- Is student (boolean)

Print all with their types.

**Exercise 2: Type Challenge**
\`\`\`python
a = "10"
b = 10
c = 10.0
\`\`\`

- What are their types?
- Can you add a + b? Why/why not?
- Can you add b + c? Why does it work?

**Exercise 3: Conversion**
Read a user's age as string, convert to int, add 10 years, print result.


## 2. Strings (From Lesson 2)

**Exercise 4: Text Processing**
\`\`\`python
text = "  Hello, World! HELLO  "
\`\`\`

- Remove whitespace
- Convert to lowercase
- Count how many times "hello" appears (after lowercase)
- Replace "world" with "python"

**Exercise 5: Slicing**
Find the file extension of "document.pdf" using slicing.

**Exercise 6: Email Validator**
Check if an email contains "@" and ends with ".com"


## 3. Numbers (From Lesson 3)

**Exercise 7: Calculator**
Read two numbers, perform +, -, *, / and print.

**Exercise 8: Temperature**
Convert 25 C to F using F = (C * 9/5) + 32

**Exercise 9: Percentage**
Calculate what percentage 25 is of 200.
