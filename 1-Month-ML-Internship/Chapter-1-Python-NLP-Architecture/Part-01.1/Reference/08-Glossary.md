# Part-01.1 Glossary

## OOP Terms
- **Class**: Blueprint for objects
- **Instance**: Specific object created from class
- **Method**: Function inside a class
- **Attribute**: Data variable in class
- **Inheritance**: Child class extends parent
- **Polymorphism**: Same method, different behavior
- **Encapsulation**: Hide internal details

## Advanced Terms
- **Decorator**: Function modifying another function
- **Generator**: Memory-efficient iterator
- **Context Manager**: Resource management (with statement)
- **Exception**: Error handling object
- **Decorator Stack**: Multiple decorators on one function
- **Closure**: Function capturing outer scope

## Data Structures
- **Counter**: Frequency counting
- **defaultdict**: Dictionary with default values
- **namedtuple**: Lightweight class
- **deque**: Double-ended queue

## Performance Terms
- **Big O**: Algorithm complexity measurement
- **Profile**: Analyze code performance
- **Benchmark**: Compare execution times
- **Generator Expression**: Lazy evaluation
